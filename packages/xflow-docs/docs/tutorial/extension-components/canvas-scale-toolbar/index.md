# ScaleToolbar 画布缩放工具栏

在垂直布局的工具栏上提供`放大画布`,`缩小画布`,`1:1比例` , `缩放到画布大小` 这 4 个常用的画布缩放操作。

# 何时使用

- 需要 `放大画布`, `缩小画布`, `1:1比例` , `缩放到画布大小`画布缩放功能时
